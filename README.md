# Nova – D2C E-commerce System

A case study focused on designing an end-to-end Direct-to-Consumer (D2C) e-commerce platform with structured business logic, user flows and operational workflows.

This project covers the complete customer journey — from product discovery to checkout and order fulfillment — with a strong emphasis on conversion flow, usability and system behavior.

It also includes admin-side operations for managing products, orders, inventory and promotions, ensuring seamless coordination between customer experience and backend operations.

The system is designed to handle real-world scenarios such as inventory constraints, order state transitions and edge cases with a focus on scalability and operational clarity.
